Weights & Styles
Only two weights. No italic. No oblique. Never set font-weight: 700 on Fagies.
WeightValueFontUsage
Regular 400Fagies All display — H1, prices, blob button labels, mega ticker, footer brand name
Regular 400Banter Grotesk Body copy, paragraphs, captions, descriptions, plain nav text
Semibold 700Banter Grotesk H2, H3, secondary button labels, nav links, footnotes bold, badges, overlines
/* Rule: Fagies → always 400 */ /* Rule: Banter Grotesk 400 → body text, descriptions */ /* Rule: Banter Grotesk 700 → anything that needs to shout — headings, buttons, labels */ /* Never: */ /* font-family: 'Fagies'; font-weight: 700; ← no semibold Fagies exists */ /* font-style: italic; ← no italic defined for either font */

Spacing Scale
4px base unit. All margin, padding, and gap values must snap to this scale. No arbitrary values.

--space-624px — grid gutter (desktop), ticker text padding, margin-bottom overline→H1
--space-832px — section side padding, card padding (desktop), margin-bottom H1→body
